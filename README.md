# Decision Tree from Scratch

This project implements a decision tree model from scratch to better understand the core logic of classical machine learning algorithms, including tree construction, splitting criteria, and prediction.

## Project Materials

- [Project report / slides (PDF)](./Decison%20%20tree%20tree%20structure%20in%20machine%20lerning.pdf)

## Overview

This project explores the structure and implementation of decision trees in machine learning. It focuses on understanding how decision trees split data, represent hierarchical rules, and perform classification.

## Highlights

- Implemented a decision tree from scratch
- Explored tree structure and splitting logic
- Studied how decision rules are formed recursively
- Built the project as a hands-on machine learning exercise

## Repository Contents

- Source code for the decision tree implementation
- Supporting project PDF / presentation
- Related notes and experiments

## Motivation

I built this project to gain a deeper understanding of classical machine learning methods by implementing a decision tree directly rather than only using a library implementation.
